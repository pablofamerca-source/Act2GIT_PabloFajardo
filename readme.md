# Actividad 2 - GIT y GitHub

Este proyecto forma parte de la asignatura **Entornos de Desarrollo** del IES Jesús Marín. El objetivo es practicar el uso de Git y GitHub mediante la creación de una página web básica y la gestión de ramas.
## Contenido
- Página principal con cabecera, menú (Inicio, Horario) y footer.
- Página de **Horario** con una tabla de clases.
- Página **Sobre Mí** con información personal.
- Etiquetado del repositorio como `V1.0`.
## Tecnologías utilizadas
- HTML
- Git y GitHub
